1 Get ready to code

  1. Open Notepad++ (Note the ++ at the end) by clicking on the Start menu, then typing Notepad++ into the search bar and clicking on the Notepad++ application when it comes up (look for an icon with a chameleon on it)
  2. Save your empty HTML file
    1. Click File, then Save
    2. Click Desktop on the left to save it on your desktop
    3. Name it with your first name and .html file extension (for example, Rebecca.html)
    4. Select All Files from the Save as type dropdown
    5. Click Save
    Pro tip! Save time by pressing CTRL + S in Notepad++ to save your file.
  3. Open Chrome by double-clicking the icon on the desktop, or by clicking on the start menu and typing Chrome into the search bar and clicking on the Chrome application (you can also use another browser, such as Firefox)
  4. Open your HTML file in Chrome
    1. Click the red, yellow and green wheel Chrome menu button in the top left of the window
    2. Click the arrow next to New Tab, then click the Open File button when it appears
    3. When the file browser window appears, click on Desktop and double-click on your HTML file
    4. You should now see a blank page in Chrome, and the URL bar (where you type web addresses) should have your HTML filename in it

This is what your screen should more or less look like

2 Try the basic template

  1. Click on: basic template - and download the file it will look just like a regular webpage
  2. Open the file in Notepad++
  3. Fill in the template by looking for instructions like IMAGE URL GOES HERE and replacing it with something like http://www.example.com/image.jpg
  4. See what your page looks like by switching to the Chrome window and refreshing the page by pressing CTRL + R
    Pro tip! Do this often as you make changes to see how your changes affect the look of the page
  5. Try out different tags like <h1> or <img> and see what you can do! Check out the manual handout for more detailed info on each tag.
    Pro tip! Check out the Get Creative sections below to see how to add links, images, and videos
  6. Don't forget to save your work!

3 Try the advanced template

  1. Create a new HTML file by clicking on File then New in Notepad++
  2. Save your empty HTML file by following the steps in Part 1, but make sure to give it a different name, such as Rebecca2.html
  3. Open your HTML file in Chrome by following the steps in Part 1
  4. Open the advanced template - it will look just like a regular webpage
  5. Copy the source code by right-clicking on the template page and choosing "View Page Source" or "View Source" from the menu, then selecting all the code (use CTRL + A like before) and copying it by pressing CTRL + C
  6. Paste the source code into Notepad++ by clicking inside the Notepad++ window and pressing CTRL + V to paste
  7. Fill in the template by looking for instructions like IMAGE URL GOES HERE and replacing it with something like http://www.example.com/image.jpg
  8. See what your page looks like by switching to the Chrome window and refreshing the page by pressing CTRL + R
    Pro Tip! Do this often as you make changes to see how your changes affect the look of the page
  9. Have fun making your webpage! Try copying sections to add more images, or change the text colour using the <font> tag.
    Pro Tip! Check out the "Get Creative" sections below to see how to add links, images, and videos
  10. Don't forget to save your work!

4 Upload your work

  1. Go to the upload page
  2. Sign in using the username and password shown in your uploading handout
  3. Upload your file by clicking "Choose File", selecting your file from the desktop, and clicking "Upload"
    Pro Tip! Do this step twice to upload both of your pages so that you can access them from home

Get Creative

How to change the font size

  1. Add a font tag to the body of your HTML like so: <font size="7">My re-sized text here</font> and replace 7 with a number from 1 to 7 (where 1 is the smallest, and 7 is the biggest), and My re-sized text here with the text you want to be big or small!
    • Tip: Remember that you can nest tags inside one another, so you can do things like use both <font> and <b> tags to make text both a different size and bold
  2. For more information on changing the font size, click here

How to change the font colour

  1. Choose a colour and copy its hex code by going to an online colour picking tool, finding a colour you like, and copying the six-digit code that represents it, which looks like #aaaaaa, except that the a's can be letters (from a-f) or numbers
  2. Add a font tag to the body of your HTML like so: <font color="#aaaaaa">My colourful text here</font> making sure to replace aaaaaa with the hex code you copied in the first step, and My colourful text here with the text you want to be colourful!
    • Tip: You can add multiple attributes (like size and color) to the same <font> tag! For example: <font size="7" color="#ff0000">My big red text here</font>
  3. For more information on changing the font color, click here

How to add a link

  1. Copy a website URL by opening the webpage in a new tab, then selecting the URL in the browser's bar and pressing CTRL-C to copy it
  2. Add an anchor tag to the body of your HTML like so: <a href="http://www.example.com/page.html">My Link Text</a> making sure to replace http://www.example.com/page.html with the URL you copied in the first step, and My Link Text with what you want the link to say
  3. For more information on adding links, click here

How to add an image

  1. Copy an image URL by looking up an image (try Google Images), then right-clicking on the image and choosing "Copy Image URL" from the menu
  2. Add an image tag to the body of your HTML like so: <img src="http://www.example.com/image.jpg"> making sure to replace http://www.example.com/image.jpg with the URL you copied in the first step
  3. For more information on adding images, click here

How to add a YouTube video

  1. Go to YouTube and choose a video
  2. Copy the embed code by right-clicking on the video and choosing "Copy Embed Code"
  3. Paste the embed code into the body of your HTML file
  4. For more information on adding videos, click here: here
